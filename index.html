maroc-news by rguigsaad

RSS Feed Reader in JQuery Mobile Of some Moroccan websites

Projects that uses JQuery Mobile + PhoneGap. It aims to offer a simple and user-friendly interface to access Rss feed of some of the most famous news websites in Morocco.

Install

Import the project on Eclipse + run => "Run as Android Application" and choose the device you want to install it on.
